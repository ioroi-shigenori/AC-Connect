# AC-Connect

　須藤先生の話を受けて、お互い知りたいのではないかと思います。
企業の人は何を知りたいのか分かりませんが、大学からは以下のこと
が知りたいと思いました。
　　企業の人が考える
　　　・大学で学んでおいて欲しいこと
　　　・これだけは譲れない、身に着けておいて欲しい言動（マナー）
　　　・企業では時間がかかりすぎて教えられないけど、身に着けて
　　　　いると助かる知識・技術
    
これからの学生は、AIに頼れば課題などは簡単に
出来てしまうので、ますます勉強しなくなると思います。
（勉強する意味がないとすら考えている感があります）

起業の人も、そういう心配はしていると思いますし、
だからといって、どうしたらいいという
答えも持っていないと思いますけども、

「AI時代における大学教育の在り方」を
起業と連携して構築していきたい、ということで協力関係を築く。

A 学部学科の編成（とても大変）
B カリキュラムの検討（Aほどではないけど大変）
C 現状の授業の内容変更（シラバス）

Cくらいからスタートするならば動きやすいのではないでしょうか

はい．これから先の重要課題だと思います．
いきなり大問題を解くことはできなので，
我々のできる部分を見つけて，実績を
作るが良いのでは．C案を具体的に
つめてはどうでしょうか．

企業への講演・授業・インターン依頼
は，ほとんどの場合に，すべてを企業に
おまかせです．企業側も大学から頼まれれば
NOと言えないので，定番の内容でこなし
ます．機械系とか，電気系は手を動かす
仕事がいくらでもあるので，ネタに困り
ませんが情報系は大変です．

ネタのところまで産学連携でやろうねと
いうのが，コーオプ教育となります（はず）．

このネタで次の情処全国大会を飲み会遠征
にするぐらいの計画で，実際の作業に入って
はどうでしょうか．
